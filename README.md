## 📖 About The Project

**NeuroCardia AI** is an advanced, web-based platform designed to assess and predict the risk of multiple types of strokes using intelligent models. The application features a stunning, interactive 3D user interface to provide a premium user experience while delivering critical health intelligence.

### ✨ Key Features

- **Interactive 3D Visualizations**: Built with `@splinetool/react-spline` and `Three.js` to display immersive Brain and Heart models.
- **Risk Assessment**: A dedicated module to calculate and predict multi-stroke risks based on user inputs.
- **Problem Statement Analysis**: Clear, educational sections explaining the unseen risks of cardiovascular and neurovascular events.
- **Modern Tech Stack**: Fast and responsive, built using React, TypeScript, and Vite.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- `npm` or `yarn`

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Mithlesh-95/Web-Based-Multi-Stroke-Risk-Prediction-System.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Web-Based-Multi-Stroke-Risk-Prediction-System
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173` to view the application.
